2. modules/email_digest.py (9.9 KB)
   └─ Function: send_daily_digest(jobs: list[dict], stats: dict, run_date: str) -> bool
      • Sends HTML email via Gmail SMTP
      • Config from environment: GMAIL_SENDER, GMAIL_APP_PASSWORD, DIGEST_RECIPIENT
      • Email subject: "Job Hunt Digest — {date} | {applied} Applied | {action_required} Need Your Attention"
      • Email sections:
        - Header with date
        - 4 stat cards (Searched, Suitable, Applied, Action Required)
        - "✅ Applied Today" table with Role, Company, Source, Score
        - "⚡ Your Action Required" table with apply URLs (clickable)
        - Footer with link to dashboard
      • Inline CSS for email compatibility
      • Returns True/False with logging

   ├─ Function: setup_logging() -> logger
   │  • Console handler (INFO level)

   │
   ├─ Function: run_agent() -> bool
   │  • Loads .env
   │  • Initializes JobDatabase
   │  • Fetches raw jobs
   │  • Filters and deduplicates
   │  • Saves to DB
   │  • Scores jobs (only status=="found")
   │  • Generates cover letters (only status=="suitable")
   │  • Runs apply engine
   │  • Generates and saves dashboard
   │  • Sends email digest
   │  • Saves run stats
   │  • Logs comprehensive summary
   │  • Graceful error handling (no crash on individual job failures)
   │
   ├─ Function: schedule_daily_run()
   │  • Uses schedule library
   │  • Gets DAILY_RUN_TIME from env (default: 08:00 IST)
   │  • Runs run_agent() on schedule
   │  • Checks every minute
   │  • Graceful Ctrl+C handling
   │
   └─ CLI with argparse:
      • --run: Run agent immediately and exit
      • --schedule: Run on daily schedule
